# Beispiel Irregulare Verben - Ankidroid
AnkiDroid kann heruntergeladen werden unter: https://play.google.com/store/apps/details?id=com.ichi2.anki&hl=en-US

1. Einen Stapel erstellen (dort befinden sich alle Karten zu einem Fach oder Theme; es können widerum Unter-Stapel erstellt werden (Bsp.: Fach > Thema))
![[Screenshot_20260417_172313_AnkiDroid.jpg|310]]
2. Einen Stapel benennen und abspeichern
![[Screenshot_20260417_172326_AnkiDroid.jpg|311]]
3. Lange drücken auf den Stapel um neue Karteikarten hinzuzufügen
![[Screenshot_20260417_172345_AnkiDroid.jpg|321]]
4. Dialog öffnet sich und nun kann wie eine ganz normale Karteikarte Inhalt hinzugefügt werden durch drücken auf den Hacken
![[Screenshot_20260417_172417_AnkiDroid.jpg|300]]
5. Nun kann im Hauptfenster auf den Stapel gedrückt werden und sofort mit dem Lernen begonnen werden
![[Screenshot_20260417_172427_AnkiDroid.jpg|270]]
6. Ankis Haupt-Feature ist, dass repititives lernen unterstützt wird. Mithilfe von ehrlichen Einschätzen werden Karteikarten so oft wiederholt bis sie vollständig gelernt sind. Wichtig dabei ist, dass man häufig (täglich!) kurz in die Anki App geht um die Karten zu üben, die wieder an der Reihe sind. Anki regelt das üben automatisch und beurteilt wann eine Karte wieder geübt werden muss
![[Screenshot_20260417_172432_AnkiDroid.jpg]]
